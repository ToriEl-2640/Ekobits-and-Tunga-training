6. To jump between words in the terminal,Use Option and the left arrow to move back and use Option with the right arrow to move forward down the line.

7. To get to the end of a line in terminal,use [Ctrl] + [E] or End

8. To move your cursor to the beginning, use [Ctrl] + [A] or Home.

9.  To delete a word withoutpressing backspace multiple times in terminal,in front of the cursor, use [Ctrl][W]. To delete the word after the cursor, use [Alt][D].

10. A Terminal is like a parent to a Shell because a terminal is a program which runs a shell giving us a command line interface to interact with the computer while a shell is a program that processes commands and returns output.

11. An Absolute path specify a path starting from the root directory.

12. A Relative path specify a path based on current location.

13. A Flag is an entry that modifies the behaviour of a command.
e.g. -a, -l, -r.

14.The -r and -f flags with the rm command removes an entire folder with files.
