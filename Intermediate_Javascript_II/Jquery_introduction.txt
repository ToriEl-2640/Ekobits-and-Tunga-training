1. JQuery is a lightweight Javascript library that compiles many codes in a single line and makes it executable. It makes it much easier to use JavaScript on a website.
2. An HTML element is a document component while a JQuery object is used to run an HTML element.
3. The dollar($) sign in JQuery is a shortcut that is used to distinguish JQuery from JavaScript.
